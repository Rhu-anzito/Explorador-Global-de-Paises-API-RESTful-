# Explorador-Global-de-Paises-API-RESTful-

Desenvolvimento de uma aplicação web interativa utilizando HTML, CSS e JavaScript para consumir e exibir dados de países.

Implementação de chamadas assíncronas (async/await e fetch API) para buscar informações (nome, capital, região, população e bandeiras) de uma API RESTful externa (RestCountries API).

Criação de uma funcionalidade de pesquisa dinâmica (onkeyup e .filter()) que permite ao usuário filtrar a lista de países em tempo real.

Foco em tratamento de erros (try...catch...finally e !response.ok) e feedback visual (mensagens de loading e error).

Utilização de manipulação do DOM para renderizar dinamicamente os cartões de país (createCountryCard) e gerenciar estados de carregamento.
